# Ruby CLI Template

## (1) Preparation

Go through `Gemfile` and uncomment gems you want to include

## (2) Installation

```sh
bin/install
```

## (3) Usage

```sh
bin/console
bin/cli help
```

### (3.1) Example

```sh
bin/cli example
```

## (4) Testing

```sh
rspec --init
# or
bundle exec rspec --init
```
